唤醒你的家庭

Home Assistant 是一个基于Python 3开发的开源家庭自动化平台。可以跟踪和控制家庭中的所有设备，并实现自动化控制。 可以完美的运行在Raspberry Pi。

    控制

    通过单一的，适配移动设备的界面控制你的所有设备

    Home Assistant 可以让你控制你的所有设备，而不用把任何数据存储在云端。我们希望保护您的隐私。

    自动化

    设置一些先进的自动化控制规则让你的家庭智能化

    你是否想 …

    • 在家时，太阳下山了，灯会自动打开
    • 晚上到家时，灯会自动打开
    • 准备看电影时，灯会自动熄灭
    • 不在家时，灯被打开，你能收到通知
